### 상호 간섭 분석 (Interaction Analysis)

두 실험을 동시에 진행할 때 발생할 수 있는 데이터 오염을 주의해야 합니다.

역효과를 최소화하고 시너지를 극대화하기 위한 가이드라인입니다.

1. **팝업의 선택권 부여:** 
    ```
    팝업이 떴을 때 '지금 풀기'와 '나중에 풀기(강의 종료 후)'를 선택할 수 있게 하세요. 하지만 '나중에 풀기'를 선택하면 "완강 시 100% 달성을 위해 꼭 필요해요"라는 문구를 띄워 방안 2와 연결합니다.
    ```
2. **진행률 가시성 개선:**
    ```
    강의 화면 옆 사이드바에 `[영상 90% + 퀴즈 10%]`라는 배분을 명확히 표시하여 유저가 규칙을 미리 인지하게 합니다. (갑작스러운 거부감 방지)
    ```
3. **데이터 분리 분석:** 
   ```
   전체 유저를 4개 그룹(A: 기존, B: 팝업만, C: 진행률만, D: 둘 다)으로 나누는 (다변량 테스트)를 고려해 보세요. 샘플 수가 적다면 B와 C를 순차적으로 테스트하는 것이 안전합니다.
   ```
